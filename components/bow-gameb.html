The Bow Game

Unity, C#, Unity XR Toolkit The bow game is a VR Archery Simulator prototype. It features moving animal sprites as targets rather than 3d objects. This project was developed over the course of 3 weeks with a team of 5 people.

Project Responsibilities

- Programmed movement system for creature targets
- Contributed to bow and arrow interaction code
- Familiarized myself with XR Interaction Toolkit
