RushRushVR

Unity, C#, Oculus VR SDK RushRushVR is a collection of VR microgames, with gameplay inspired by titles such as WarioWare. Players rapidly cycle between scenes with simple tasks to complete, but with a quickly decreasing timer putting pressure on them. This project was created in about 6 weeks as a team of 5 developers.

Project Responsibilities

- Conceptualized overall game design and flow, as well as several microgame designs
- Developed interactions, game flow, and level layout for 'Bug Catcher' microgame
- Developed interactions, game flow, and level layout for 'Ice Cream Stack' microgame
