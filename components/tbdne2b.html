'This (Blank) Does Not Exist'

Unity, C#, JavaScript, Google Firebase TBDNE, or 'This (Blank) Does Not Exist', is a wordle-esque daily puzzle game where players try to distinguish between real and AI-generated photos. This was created during my Summer 2023 internship at Retora Games with a team of 2 developers and 1 designer.

Project Responsibilities

- Created back-end Firebase database for game metrics collection and game data analysis
- Integrated Firebase back-end with live website using JavaScript API
- Developed locally stored player statistics page for personalization
- Contributed to UI/UX design
