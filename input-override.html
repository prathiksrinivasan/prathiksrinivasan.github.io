Input Override

Project Roles: Game Design, Programming, Illustration
Input Override is a game created in 72 hours for the 2023 GMTK Game Jam, based on the theme "Roles Reversed". Taking inspiration from traditional fighting games, this game has the player take on the role of the 'input interpreter', observing the motions input by an unseen player and choosing the correct move from the move list.
