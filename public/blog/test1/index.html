Jul 2024

UE5 C++ Python UI Design

In my Spring and Summer internships at Brunwick’s BI (Boating Intelligence) Design Lab, I helped create a futuristic boating simulation to showcase the company’s future endeavors at CES 2025.

Project Overview

I was part of the Computer Graphics team at the BI Design Lab, and our primary task for the spring and summer was to develop an interactive futuristic boating experience using Unreal Engine 5. The various elements of the experience are intended to showcase future developments in technology that Brunswick is pursuing to companies and investors attending CES 2025. This was my first role as a developer on a large-scale Unreal Engine project, and it was interesting to learn about the different considerations taken into account for a physical full-scale demonstration as opposed to traditional game development with the engine.

Responsibilities

  • Collaborated with UI/UX Designers and artists to translate visual layouts and planned interactions to HMI user interface in Unreal Engine
  • Developed and maintained a platform-agnostic communication protocol to share data wirelessly accross multiple PCs, Android displays, and various microcontrollers
  • Helped translate our custom UI components and frameworks into a plugin that could be utilized in future UI development project
  • Evaluated new tools for UI design such as the RIVE plugin, reporting performance and compatibility concerns
  • Created and manage weekly android builds with latest features for frequent testing

Skills

  • Unreal Engine 5 Prior to this project, my experience with Unreal Engine was limited to student projects. I was able to gain experience with this engine and it’s intricacies through contributing to the development of a structured project.
  • C++ plugin development I was responsible for developing a C++ plugins that could allow us to utilize features that we had developed in other projects, focusing on extensibility and reusability.
  • Blueprint Scripting In this project, we used visual scripting to prototype new features and interactions for both UI components and actors.
  • UMG and Common UI To develop the UI for this project, we made extensive use of Unreal’s UI development tools, allowing me to gain experience with using C++ and the Unreal Editor to develop interfaces according to a desiger’s specifications
  • Perforce Through this project, I learned the Perforce/P4V workflow, becoming familiar with a commonly used source control solution for Unreal.
