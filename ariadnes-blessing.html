Ariadne's Blessing

Project Roles: Lead Programmer, Technical Designer
Ariadne's Blessing is a puzzle horror game, where the player must escape a procedurally generated maze. This is a semester-long project created in Unity with the University of Illinois Urbana-Champaign GameBuilders Studio, a student-led game developer team of over 20 developers. I am the programming lead, responsible for managing the programming team and prototyping new features.
Download on itch.io!

Project Responsibilities

Lead Programmer - As the lead programmer, I was responsible for specifying and assigning tasks to the rest of the programming team. We used Jira to organize feature requests and keep track of blocking tasks.

Integration  - I reviewed and approved pull requests for other programmer's features, integrating them into the main branch and assembling new inclusions into the main game scene. 

Gameplay Programming - I prototyped new features to add to the gameplay, such as the item system that allows players to use certain powerups to help them navigate the maze.
